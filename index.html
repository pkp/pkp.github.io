PKP Code Resources

Welcome! The Public Knowledge Project maintains Open Journal Systems, Open Monograph Press, Open Conference Systems, and more. Here you'll find a list of code repositories you can download, modify, and contribute to.

Application Code Repositories

  • Open Journal Systems
  • Open Monograph Press
  • Open Conference Systems
  • Open Harvester Systems

Our applications make use of a shared library called the PKP Web Application Library.

Other Projects

  • XML Parsing Service: An automatic XML parsing service to convert scholarly documents into JATS XML

Tools

  • OJS Stats: Crunch statistics about OJS usage.
  • Vagrant Images: Vagrant configurations for VM-based installations of PKP software.
  • Scholarly Bot List: A list of user agents that should be considered robots when tracking readership statistics.

Documentation

  • OMP User Guide

Plugins

  • Google Analytics: Track reader statistics.
  • ORCiD Profile: Integrate ORCiD authentication with registration and user profile.
  • PDF.js Viewer: Deliver PDFs in OJS using the PDF.js reader.
  • Custom Block Manager: Create custom sidebar blocks through the web interface.
  • Static Pages: Create custom static pages through the web interface.
  • TinyMCE: Add support for rich text editing to PKP applications with TinyMCE.
  • Email Logger: Log emails in the database for auditing and testing.
  • Translator: Edit translations directly from within PKP applications.
  • OJS Markup: Integrate the XML Parsing Service into OJS for automatic conversion of scholarly documents into JATS XML.
  • Default Translation: Use English-language text when translations are missing.
  • ALM: Present Article-Level Metrics for published content in OJS.
